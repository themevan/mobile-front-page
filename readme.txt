Tested up to: 4.9.6
Stable tag: 1.0.0


== Description ==

If you want to set the different front page for the mobile phone or tablet, you made a right choice, just activate this Mobile Front Page plugin. There are 4 kinds of device can be detected: iPhone, Android phone, iPad, Android tablet. 

In this initial release version, it only includes device detecting and page redirection feature. If you have any suggestions, please feel free to submit issues on this <a href=“https://github.com/themevan/mobile-front-page/issues” target=“_blank”>GitHub page</a>, and the support questions should be asked in the <a href=“https://wordpress.org/support/plugin/mobile-front-page”>plugin support forum</a>.


Credits:
The icons which are used in WordPress.org plugin page header picture and icon is designed by <a href=“https://thenounproject.com/sonataely” target=“_blank”>Elizabeth Lopez</a>.


== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory or install the plugin via Plugins page in WordPress backend.
2. Activate the plugin.
3. Go to Settings > Mobile Front Page, you can select the page for each devices.
4. When you visit your website on mobile devices, you will see the front page you have set for these devices.
